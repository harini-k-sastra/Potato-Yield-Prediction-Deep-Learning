# Potato Yield Prediction Using Deep Learning

## Overview
This project predicts potato yield using deep learning techniques and agricultural data.

## Technologies
- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Scikit-Learn

## Models Implemented
- VGG16
- VGG19
- GoogLeNet
- MobileNet
- ResNet50
- ResNet59

## Dataset Features
- Area
- Year
- Rainfall
- Temperature
- Pesticide Usage
- Yield

## Evaluation Metrics
- R2 Score
- Mean Squared Error (MSE)
- Mean Absolute Error (MAE)
